# Content Directory

Place all images and media assets in this directory.

## Required Images

When implementing a website from this template, add the following images:

1. **favicon.ico** - Browser tab icon (16x16 or 32x32 pixels)
2. **placeholder-about.jpg** - About section image (recommended: 800x600px)
3. **placeholder-feature-1.jpg** - First feature/portfolio image (recommended: 800x600px)
4. **placeholder-feature-2.jpg** - Second feature/portfolio image (recommended: 800x600px)

## Image Guidelines

- **Format:** JPG for photos, PNG for logos/graphics with transparency
- **Size:** Optimize images to < 1MB each for fast loading
- **Naming:** Use lowercase with hyphens (e.g., `team-meeting.jpg`)
- **Dimensions:** Aim for 1200px width maximum for retina displays

## Optimization Tools

- **TinyPNG:** https://tinypng.com/
- **ImageOptim:** https://imageoptim.com/
- **Squoosh:** https://squoosh.app/
